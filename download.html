How to Verify

Using PowerShell:
  1. Open PowerShell.
  2. Navigate to the directory where you downloaded the installer.
  3. Run the following command to get the hash of the file:
    Get-FileHash -Algorithm SHA256 EonDev_0.1.0_x64_en-US.msi
  4. Compare the generated hash with the one provided above.
Verifying the Certificate:
  1. Right-click on the installer file and select "Properties".
  2. Go to the "Digital Signatures" tab.
  3. Select the signature from the list and click "Details".
  4. In the "Details" window, view the certificate.
  5. Under the "Details" tab of the certificate, look for the "Thumbprint" field and compare it to the fingerprint above.
